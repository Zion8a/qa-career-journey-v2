# Part 13 – Practical Test Project

This section documents my work during the course Practical Project Work in Software Testing as part of my education as a Software Tester at EC Utbildning.

The purpose of this project is to show how I work with structured testing in a realistic project context. The focus is on test planning, exploratory testing, test case design, bug reporting, regression testing, requirement review and communication of test results.

## Course focus

This part connects to the following areas:

- Software development processes and testing
- Static testing
- Testing in agile teams
- Test techniques
- The test process
- Test reporting and communication
- Critical review of requirements
