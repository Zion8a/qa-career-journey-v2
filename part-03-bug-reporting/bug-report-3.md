# Bug Report 3

## Titel
Otydligt felmeddelande vid misslyckad inloggning

## Sammanfattning
När användaren anger fel lösenord visas ett felmeddelande som är otydligt och inte hjälper användaren att förstå vad som blivit fel.

## Förutsättning
Användaren befinner sig på inloggningssidan och har ett giltigt konto.

## Steg för att återskapa
1. Öppna inloggningssidan.
2. Ange giltig e-post eller användarnamn.
3. Ange fel lösenord.
4. Klicka på inloggningsknappen.

## Förväntat resultat
Ett tydligt men säkert felmeddelande visas, till exempel att inloggningsuppgifterna inte kunde verifieras.

## Faktiskt resultat
Felmeddelandet är otydligt och hjälper inte användaren att förstå situationen.

## Allvarlighetsgrad
Medel

## Prioritet
Medel

## Kommentar
Felet påverkar användarupplevelsen och kan skapa frustration, även om tjänsten fortfarande går att använda efter nytt försök.
