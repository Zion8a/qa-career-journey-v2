# Part 05 Newman

Det här avsnittet fokuserar på att köra API-tester från terminalen med Newman.

## Innehåll
- `restful-booker-tests.postman_collection.json` – collectionfil som kan köras med Newman
- `newman-run-notes.md` – dokumentation av den första Newman-körningen

## Syfte
Syftet med arbetet är att ta API-testning från Postman-gränssnittet till repeterbar körning i terminalen.

## Det jag tränar på
- att förstå hur Postman och Newman hänger ihop
- att köra collectioner via terminal
- att dokumentera automatiserad API-körning
- att bygga grund för CI/CD senare
